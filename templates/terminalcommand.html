Command Description
ls Lists files and directories in the current directory.
cd [directory] Changes the current directory to the specified directory.
pwd Displays the full path of the current directory.
mkdir [directory] Creates a new directory.
touch [file] Creates a new empty file.
rm [file] Deletes a file (does not hold it in the trash).
rmdir [directory] Deletes an empty directory.
cp [source] [destination] Copies a file from source to destination.
mv [source] [destination] Moves a file from source to destination (can also be used to rename it).
echo [text] > [file] Creates a file and inserts text into it.
cat [file] Displays the content of a file.
chmod [permissions] [file] Changes the permissions of a file (e.g., chmod 755).
chown [user:group] [file] Changes the owner and group of a file.
sudo [command] Executes a command as the system administrator.
top Displays an interface with a list of active processes and system usage.
kill [PID] Terminates a process based on its Process ID.
df Displays information about free and used disk space.
du [directory] Displays the disk space used by a directory.
ssh [user]@[host] Connects to an external server via SSH.
curl [URL] Performs an HTTP request to a URL and displays the content.
nano [file] Opens a simple text editor in the terminal.
vim [file] Opens an advanced text editor in the terminal.
grep [pattern] [file] Searches for specific text in a file.
find [directory] -name [filename] Searches for files by name within a specific directory structure.
history Displays the command history entered in the terminal.
tar -cvzf [archive.tar.gz] [files] Creates a compressed archive file from specified files.
tar -xvzf [archive.tar.gz] Extracts files from a compressed archive.
ssh-keygen Generates SSH keys for secure authentication.
ping [host] Sends ICMP requests to a server and displays the estimated round-trip time.
ifconfig Displays network information and IP addresses.
wget [URL] Downloads a file from the internet.
scp [file] [user]@[host]:[destination] Copies a file to an external server via SSH.
chmod +x [file] Allows execution of a script or program.
uptime Displays the system's uptime.
who Displays information about users currently logged into the system.
