Command Description Example
flask run Runs the Flask development server flask run
flask shell Opens an interactive Python shell with the Flask application context flask shell
flask routes Displays a list of all URL routes defined in the application flask routes
flask shell --ctx Opens a Python REPL with access to the main server context (not just the development context) flask shell --ctx
flask --app APP_NAME run Runs the explicitly defined Flask application specified by APP_NAME flask --app myapp run
flask --version Displays the installed Flask version flask --version
flask --help Displays help information about Flask commands, including available commands, options, and arguments flask --help
flask --app APP_NAME routes Displays the list of routes for a specific Flask application specified by APP_NAME flask --app myapp routes
flask --app APP_NAME shell Opens an interactive Python shell with the context of a specific Flask application specified by APP_NAME flask --app myapp shell
flask --app APP_NAME --version Displays the Flask version installed in a specific Flask application specified by APP_NAME flask --app myapp --version
flask --app APP_NAME --help Displays help information about commands for a specific Flask application specified by APP_NAME flask --app myapp --help
