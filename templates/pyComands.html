Command Description
python Runs the Python interpreter.
python script.py Runs a Python script.
pip install package-name Installs a Python package using pip.
pip uninstall package-name Uninstalls a Python package using pip.
virtualenv venv Creates a virtual environment named 'venv'.
source venv/bin/activate Activates the virtual environment.
deactivate Deactivates the virtual environment.
python -m venv venv-name Creates a virtual environment with a specific name.
pip freeze Lists installed packages in the current environment.
python -c "print('Hello, World!')" Runs a Python one-liner from the command line.
python -m http.server port Starts a simple HTTP server on the specified port.
python -c "import module_name" Runs Python code from the command line using a specific module.
python -m pydoc module_name Displays documentation for a Python module.
python -m unittest discover Runs the unittest test discovery.
python -m timeit "code" Measures the execution time of a specific code snippet.
python -m site Displays information about Python's site-packages directory and configuration.
python -m ensurepip Ensures that pip is installed or upgrades it if necessary.
python -m zipfile -c archive.zip file1 file2 file3 Creates a ZIP archive using the zipfile module.
python -m sqlite3 database.db Opens an SQLite database and enters the SQLite prompt.
python -m trace --trace script.py Runs a trace on a Python script.
