Command or Property Explanation
Label Widget designed for displaying text.
Button Widget intended for user interface applications, including a clickable button.
TextInput Widget that allows user text input.
Image Widget designed for displaying images.
Widget Basic class from which all widgets in Kivy inherit. Allows you to create and customize personalized widgets.
Builder Tool for loading .kv files to define the user interface separately from the code.
kv The description language of Kivy, used to define user interfaces and their relationships.
AnchorLayout Layout that allows you to anchor an item to a specific location in the display and change its position based on window size changes.
GridLayout Layout for arranging display systems in rows and columns, just like a table.
Clock Class that provides tools for managing timing and creating timed functions in Kivy.
ScrollView Widget that allows you to display large content outside the display frame through scrolling.
Slider Widget for controlling numeric values by moving a slider within a range.
ProgressBar Widget used for displaying progress in a task or process.
Switch Widget intended to toggle between two states, like a switch.
Spinner Widget designed to display options in a dropdown list.
Canvas Widget that allows you to draw custom graphics on other widgets using loops, painting, and more.
